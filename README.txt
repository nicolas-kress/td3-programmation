Elevators (TD3 Programmation)
------------------------------
September/October 2025 - L3 Informatique - ENS Paris-Saclay

Description: 
  Programming exercise in context of a course at ENS Paris-Saclay. 
  Branch 'main' fills project criteria. 
  Branch 'plague' upgrades the project to a pandemic simulator.
  Executable file: 'mstring.cpp'
  Parameters are set at the top of 'mstring.cpp' file or 'mstring.h' header file.


Known errors/bugs: 
  No known errors as of 10/07/2025.
  To help debugging, enable dumping by changing the value of 'DUMP' to '1' in file 'mstring.cpp'.
